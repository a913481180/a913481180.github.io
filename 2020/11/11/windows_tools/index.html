PowerToys

是一款实用工具的集合，它并不是新事物，Windows 95就曾有过这款工具，只是Windows在原来的启发之下开发了这款针对Window 10系统更加高效的版本。
目前PowerToys包括如下7项功能，每一项都非常实用，它们分别是：

  • FancyZones
  • File Explorer Add-ons
  • Image Resizer
  • Keyboard Manager
  • PowerRename
  • PowerToys Run
  • Shortcut Guide
